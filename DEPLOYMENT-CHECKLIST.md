## Build Process
The `build.sh` script:
1. Installs frontend dependencies
2. Builds frontend with Vite
3. Installs backend dependencies
4. Copies frontend build to `backend/public/`
5. Backend serves frontend from `backend/public/`

## Troubleshooting

### Frontend shows blank page
- Check browser console for errors
- Verify `backend/public/index.html` exists
- Check that vendor chunk is loading correctly

### "React is not defined" error
- This should be fixed by the simplified Vite config
- If still occurring, check that vendor chunk loads before main chunk

### Port binding timeout
- Ensure PORT environment variable is set to 10000
- Check that server is listening on 0.0.0.0:10000

### Database connection errors
- Verify Supabase credentials in environment variables
- Check that tables exist in Supabase
- Run `/health/db` endpoint to test connection
